# 生成AIパスポート Quest

生成AIパスポート学習向けのスマホ対応クイズアプリです。

## 使い方

`index.html` をブラウザで開くと利用できます。

GitHub Pagesで公開する場合は、このフォルダ内のファイルをリポジトリのルートに置き、Pagesの公開元を `main` ブランチの `/ (root)` に設定してください。

## 内容

- オリジナル練習問題 60問
- 第1章から第5章まで各12問
- 単一選択、複数選択
- 3分10問、60問模試、弱点復習、ランダム15問
- 進捗はブラウザのlocalStorageに保存

## 注意

このアプリの問題は、GUGA公開情報を参考にしたオリジナル問題です。実際の試験問題や過去問を転載したものではありません。
